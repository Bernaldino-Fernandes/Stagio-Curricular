# CRUD Simples de Tarefas (Python)

## Descrição
Aplicação simples de linha de comando para gerenciar uma lista de tarefas.
Permite:
- Adicionar tarefa
- Listar tarefas
- Marcar como concluída
- Remover tarefa

Os dados são salvos em um arquivo `tarefas.json`.

---

## Tecnologias
- Python 3.13
- JSON (para persistência)
